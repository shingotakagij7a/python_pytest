# IntTriangle

- [x] 3 つの正の整数の入力から三角形のインスタンス作成に成功する。
  - [x] 1,1,1 の場合
  - [x] 2,2,2 の場合 -> 自信がないから
  - [x] 0,0,0 の場合
  - [x] -1,1,1 の場合
  - [x] 0.1,1,1 の場合
  - [x] 1,1,2 の場合
  - [x] 1,2,3 の場合 -> 自信がないから

- [x] 三角形の周囲の長さを 3 辺の値の和として計算した整数値を返す。
  - [x] 1,1,1 の場合

- [x] 三角形の面積を 3 辺の値からヘロンの公式で計算して小数点第 2 位を四捨五入した小数値を返す。
  - [x] 3,4,5 の場合
  - [x] 5,12,13 の場合 -> 自信がないから
  - [x] 1,1,1 の場合
